    区域¶

    在现实生活中，我们的家中可能有客厅，厨房，洗手间等房间。我们可以在网关中创建对应的区域，然后将设备和情景设置在这些区域中进行统一的管理和控制。

    获取区域信息¶

    网关中已默认添加了主区域，主区域下包含子区域。

    • Android
    AreaApi.getFloorList(hostId, new MqttCallback<AreaPrimaryList.Recv>() {

    @Override
    public void onSuccess(AreaPrimaryList.Recv recv, int callBackMark, String errorCode) {
        List<AreaPrimaryList.Recv.PrimaryArea> primaryAreaList = recv.getConfig();
        for (AreaPrimaryList.Recv.PrimaryArea primaryArea : primaryAreaList) {
            // 主区域 id
            primaryArea.getId();
            // 主区域名称
            primaryArea.getName();

            List<AreaPrimaryList.Recv.PrimaryArea.SecondaryArea> secondaryAreas = primaryArea.getSecondaryAreaList();
            for (AreaPrimaryList.Recv.PrimaryArea.SecondaryArea secondaryArea : secondaryAreas) {
                // 子区域 id
                secondaryArea.getId();
                // 子区域对应的父区域 id
                secondaryArea.getParentId();
                // 子区域名称
                secondaryArea.getName();
            }
        }
    }
});

    创建、修改区域信息¶

    原则上网关不允许创建主区域，需要使用提供的主区域，在其下创建子区域。

    • Android
    // 创建区域
AreaApi.AreaBuilder builder = new AreaApi.AreaBuilder(parentId) // 主区域名称
    .setName("客厅") // 子区域名称
    .setImg("00") // 图标由开发者自定义，对应自己的图标标识
    .setCreateDate(new Date());
AreaApi.editRoom(hostId, builder, new MqttCallback<String>() {

    @Override
    public void onSuccess(String s, int callBackMark, String errorCode) {
        // 创建成功
    }

    @Override
    public void onFailure(int callBackMark, String errorCode) {
        // 创建失败
    }
});

// 修改区域
AreaApi.AreaBuilder builder = new AreaApi.AreaBuilder(parentId, areaId) // 如果是修改子区域信息，保持 areaId 不变
    .setName("客厅")
    .setImg("00")
    .setCreateDate(new Date());
AreaApi.editRoom(hostId, builder, new MqttCallback<String>() {

    @Override
    public void onSuccess(String s, int callBackMark, String errorCode) {
        // 修改成功
    }

    @Override
    public void onFailure(int callBackMark, String errorCode) {
        // 修改失败
    }
});

    获取设备、情景¶

    • Android
    AreaApi.getRoomObjects(hostId, area.getId(), area.getParentId(), AreaObjectType.all, new MqttCallback<AreaAllObject.Recv>() {

    @Override
    public void onSuccess(AreaAllObject.Recv recv, int callBackMark, String errorCode) {
        AreaAllObject.Recv.Objects objects = recv.getConfig().get(0);
        List<AreaAllObject.Recv.Objects.Object> devices = objects.getDevice(); // 区域设备列表
        List<AreaAllObject.Recv.Objects.Object> scenes = objects.getScene(); // 区域情景列表
        for (AreaAllObject.Recv.Objects.Object device : devices) {
            AreaObject object = new AreaObject(AreaObjectType.device, device);
        }
        for (AreaAllObject.Recv.Objects.Object scene : scenes) {
            AreaObject object = new AreaObject(AreaObjectType.scene, scene);
        }
    }
});
