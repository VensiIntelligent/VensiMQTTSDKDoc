    情景¶

    网关的情景功能即是设置一组动作，在其中设置多个设备的控制状态，在触发情景时一起执行这些状态控制，列如回家情景，里面设置了客厅多盏灯一起开启。

    添加、修改情景¶

    • Android
    SceneApi.SceneActionBuilder sceneActionBuilder = new SceneApi.SceneActionBuilder()
    .setDeviceId(device.getDeviceId()) // 设备名称
    .setDeviceType(device.getDeviceType()) // 设备类型
    .setDeviceName(device.getDeviceName()) // 设备名称
    .setAttrId("status") // 控制参数
    .setAttrValue("on")   // 控制值
    .setOrder("0") // 顺序
    .setDelayTime("0"); // 间隔时间 秒
SceneApi.SceneBuilder builder = new SceneApi.SceneBuilder(hostId);
ArrayList<SceneApi.SceneActionBuilder> sceneActionBuilders = new ArrayList<>();
sceneActionBuilders.add(sceneActionBuilder);
builder
    .setImg("00") // 图标由开发者自定义，对应自己的图标标识
    .setName("回家情景")
    .setConfig(sceneActionBuilders);
// 编辑情景的话，保持情景 id 不变  SceneApi.SceneBuilder(hostId, sceneId); SceneApi.SceneActionBuilder(actionId)
SceneApi.edit(hostId, builder, new MqttCallback<String>() {

    @Override
    public void onSuccess(String s, int callBackMark, String errorCode) {
        // 创建成功
    }

    @Override
    public void onFailure(int callBackMark, String errorCode) {
        // 创建失败
    }
});
