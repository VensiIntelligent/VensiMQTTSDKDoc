    Android SDK 概述¶

    Android SDK 通过使用 eclipse/paho.mqtt.java、eclipse/paho.mqtt.android 等 MQTT 库提供与智能网关的连接能力，并且提供对网关协议的解析，方便开发者迅速接入并实现智能设备控制。

    支持的 Android 系统版本¶

    目前 SDK 只支持 Android 4.4 或以上的手机系统。

    集成步骤¶

    查看 SDK 压缩包中内容¶

    • Vensi MQTT SDK 核心包

      • libs/vensi-mqtt-sdk-proguard.jar

    • MQTT 连接能力包

      • libs/vensi-mqtt-service-java-proguard.jar

    • MQTT 连接能力包（包含 Android 实现）

      • libs/vensi-mqtt-service-android-release.aar

    SDK 文件移植¶

    复制 libs/vensi-mqtt-sdk-proguard.jar 和 libs/vensi-mqtt-service-java-proguard.jar 到工程 libs/ 目录下

    说明：Android 平台建议集成 libs/vensi-mqtt-service-android-release.aar 包，这时就不需要集成 libs/vensi-mqtt-service-java-proguard.jar。AAR 包的集成请自行查找相关教程。

    在 app/build.gradle 中添加

    dependencies {
    implementation files('libs/vensi-mqtt-sdk-proguard.jar')
    implementation files('libs/vensi-mqtt-service-java-proguard.jar')
    implementation 'org.eclipse.paho:org.eclipse.paho.client.mqttv3:1.2.5'
    implementation 'com.google.code.gson:gson:2.8.6'
}

    说明：org.eclipse.paho.client.mqttv3 和 gson 版本可以修改其最新版本，但不能小于上面的版本

    配置 AndroidManifest.xml¶

    设置相关的权限

    <!--AndroidManifest.xml-->
// 允许应用可以访问网络。
<uses-permission android:name="android.permission.INTERNET" /> 
// 允许应用获取网络信息状态，如当前的网络连接是否有效。
<uses-permission android:name="android.permission.ACCESS_NETWORK_STATE" />

    添加代码¶

    初始化¶

    // 是否开启 debug 模式
VensiMqtt.getInstance().setDebug(true);
// 用户名称，保持唯一
VensiMqtt.getInstance().setUserId(userId); 
// 公司标识（网关标识）
VensiMqtt.getInstance().setCompany(company); 
// 设置用户 OS 显示
VensiMqtt.getInstance().setOs(Build.MODEL); 
// 设置请求回调线程
Handler Handler = new Handler(Looper.getMainLooper());
VensiMqtt.getInstance().setMqttMessageThreadHandler(new IMqttMessageThreadHandler() {
    @Override
    public void execute(Runnable runnable) {
        Handler.post(runnable);
    }
});
// 设备基础配置文件缓存路径
VensiMqtt.getInstance().initDeviceTypeUtils(context.getFilesDir().getAbsolutePath()); 
// 加载设备基础配置文件
DeviceTypeUtils.getInstant().loadConfigFromAssetsFileWithResult();

    创建 MQTT 连接¶

    // 创建连接参数
ConnectOption connectOption = new ConnectOption();
// 设置连接ip或域名（例如：网关在局域网中的 192.168.1.188:1883)
connectOption.setServerUri(ip);
// 创建连接服务，并传入连接参数
IMqttService service = new JavaMqttService(connectOption);
// 开始连接
VensiMqtt.getInstance().connectMainServer(service, new OnMqttStateChangedListener() {
    @Override
    public void onStateChanged(String serverIp, int statue, Throwable case) {
        switch (state) {
            case MqttStateCode.MQTT_STATE_INIT:
                // 初始化中
                break;
            case MqttStateCode.MQTT_STATE_CONNECTING:
                // 正在连接服务器
                break;
            case MqttStateCode.MQTT_STATE_CONNECTED:
                // 连接服务器成功
                break;
            case MqttStateCode.MQTT_STATE_CONNECT_FAILURE:
                // 连接服务器失败
                break;
            case MqttStateCode.MQTT_STATE_CONNECT_LOST:
                // 与服务器断开连接
                break;
            case MqttStateCode.MQTT_STATE_CONNECT_LOST_INITIATIVE:
                // 手动断开连接
                break;
            case MqttStateCode.MQTT_STATE_USER_ID_SUBSCRIBE_SUCCESS:
                // 订阅 userId 成功（建议在此处进行网关注册，登录的操作）
                break;
            case MqttStateCode.MQTT_STATE_USER_ID_SUBSCRIBE_FAILURE:
                // 订阅 userId 失败 
                break;
        }        
    }
});

    注册网关或登录网关¶

    在连接 MQTT 成功后，可以进行注册网关->登录网关->获取登录后的 sessionId。 如果已经进行过一次注册网关（查看网关功能相关说明），后面只需要进行登录网关的操作。

    // 注册网关
HostApi.register(hostId, "123456", new IBaseCallback<HostRegister.Recv>() {
    @Override
    public void onSuccess(HostRegister.Recv recv, int callBackMark, String errorCode) {
        if ("10017".equals(errorCode)) {
            toast("等待网关管理员同意");
        } else if ("10001".equals(errorCode)) {
            String sspen = recv.getSspen();
            if ("1".equals(sspen)) {
                toast("已注册网关, 并且已授权");
            } else {
                toast("已注册网关, 但管理员未授权");
            }
        } else {
            toast("注册网关成功");
        }
    }

    @Override
    public void onFailure(int callBackMark, String errorCode) {
        if ("10026".equals(errorCode)) {
            toast("网关标识错误");
        } else {
            toast("注册网关失败 " + errorCode);
        }
    }
});

// 登录网关
HostApi.login(hostId, "123456", new IBaseCallback<HostLogin.Recv>() {
    @Override
    public void onSuccess(HostLogin.Recv recv, int callBackMark, String errorCode) {
        VensiMqtt.getInstance().setMainHost(recv.getHostId());
        toast("登录网关成功 " + recv.getSessionId());
    }

    @Override
    public void onFailure(int callBackMark, String errorCode) {
        if ("10002".equals(errorCode)) {
            toast("密码错误");
        } else if ("10025".equals(errorCode)) {
            toast("等待网关管理员同意");
        } else {
            toast("注册网关失败 " + errorCode);
        }
    }
});
